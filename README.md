## About

**blu|mark Studio** is a digital innovation studio focused on building AI-powered tools and data products across three domains:

- **Real Estate** — acquisition assistants, underwriting agents, and investor-ready analytics
- **Agritech / CEA** — controlled-environment agriculture modeling, digital twins, and climate-resilient food systems
- **Transactions & Content** — LLM-driven deal workflows, property storytelling, and narrative assets

This GitHub Pages site serves as the public portfolio hub for all projects developed under the blu|mark brand.
